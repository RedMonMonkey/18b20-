文件说明：
仿真文件中有①18b20仿真模型及自行仿真及②野火驱动代码及仿真
将①中两个文件仿真可以验证模型的时序，可修改gh18b20_test.sv文件来发送相应的命令
②中包含野火FPGA的驱动代码，该代码可以在FPGA中驱动18b20芯片，将文件夹下所有波形一起仿真可以得到波形文件夹下的图片

波形：
波形1是复位后发送cc（跳过rom）和44（温度转换）后，可以看到仿真模型的内部寄存器op_ins和rom_ins正确接收到指令
波形2是复位后发送cc（跳过rom）和be（发送全部高速缓冲区）后，可以看到正确接收指令，并且仿真模型不断发送数据，最终驱动文件的data_tmp接收到0191（十六进制），
并且data_out将该十六进制数转换为十进制25062，后三位为小数部分

18b20仿真模型说明
实现功能：
复位
rom功能：cch(跳过rom)
操作功能：44h（温度转换，并未真正实现，只是可以读取该操作码），BEh(读取全部高速缓冲区)
高速缓冲区被固定为：
	byte0_t_lsb = 8'h91;
    	byte1_t_msb = 8'h01;
    	byte2_th = 8'h4b;
    	byte3_tl = 8'h46;
    	byte4_config = 8'h7f;
   	byte5_res0 = 8'h55;
   	byte6_res1 = 8'h55;
  	byte7_res2 = 8'h55;
    	byte7_crc = 8'h55;
即温度被固定为25.0625摄氏度，其余寄存器没有意义
有需要会增加其余功能。
